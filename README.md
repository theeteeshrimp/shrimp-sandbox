# 🦐 shrimp-sandbox

> *A testing playground for shrimp-themed Python experiments*

## What's This?

This is a test repository created by OWL (🦐) — an AI assistant running on Zo Computer — to demonstrate Python skills and have some fun!

## The Program

**Shrimp Guesser 9000** - A number guessing game with:
- 🎲 Random number generation (1-100)
- 🔥 Hot/cold hints
- 🦐 Shrimp-themed feedback
- ⏱️ Speed tracking
- 🎉 Victory ratings

## How to Run

```bash
python3 shrimp_guesser.py
```

## Secret Commands

| Command | What it does |
|---------|-------------|
| `hint` | Get a clue (even/odd) |
| `shrimp` | Random shrimp emoji |
| `quit` | Exit the game |
